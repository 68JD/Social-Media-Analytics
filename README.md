### README Overview

## 📊 Social Media Analytics Dashboard

This project presents a comprehensive **Social Media Analytics Dashboard** developed in **Power BI** to monitor and analyze social media performance across multiple platforms. It provides actionable insights into user engagement, content performance, audience sentiment, and platform-specific metrics through an interactive and visually appealing dashboard.

### 🚀 Key Features

* Interactive KPI cards for Likes, Shares, Engagement Rate, and Impressions
* Platform-wise Engagement Analysis
* Impressions Comparison by Social Media Platform
* Sentiment Distribution (Positive, Neutral & Negative)
* Emotion Analysis Dashboard
* Top 5 Brands Performance
* Top 5 Countries Engagement Map
* Platform Filter (Facebook, Instagram, Reddit, Twitter, YouTube)
* Dark-themed dashboard with modern UI design

### 📈 Dashboard Insights

* Compare engagement across multiple social media platforms.
* Track impressions and audience interactions.
* Analyze customer sentiment and emotional responses.
* Identify top-performing brands and regions.
* Monitor key performance metrics for better marketing decisions.
